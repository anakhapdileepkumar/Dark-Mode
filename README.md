# Dark Mode Toggle

A simple webpage that allows users to switch between Light Mode and Dark Mode with a single button click.  
The design changes background and text colors smoothly for a better user experience.

## Features
- One-click light/dark mode switch
- Smooth color transition effects
- Built with pure HTML, CSS, and JavaScript

## How to Use
1. Download or clone this repository.
2. Open `index.html` in your browser.
3. Click the **"Toggle Dark Mode"** button to switch between themes.

## Files
- **index.html** → The webpage structure
- **style.css** → The styles for both light and dark themes
- **script.js** → The JavaScript code to handle mode switching
